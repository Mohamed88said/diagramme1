Vision & Objectifs

Vision

Devenir la marketplace universelle qui permet à toute personne ou entreprise (vendeur, prestataire, particulier) d'offrir un produit ou service, d'accroître sa visibilité et de monétiser ses compétences.

Objectifs Initiaux

  • Lancement opérationnel à Conakry (Guinée)
  • Offrir un système sécurisé (escrow + KYC) et accessible (Mobile Money + carte + wallet)
  • Réduire le chômage local en valorisant micro-services et talents
  • Offrir produits physiques, numériques et services (réservation/présentiel)

Valeur à Long Terme

Plateforme unifiée où tout se vend et s'achète — produits, services, enseignement, restauration, déplacements, etc.

Cible & Cas d'Usage

Utilisateurs

  • Acheteurs / consommateurs
  • Vendeurs (professionnels & particuliers)
  • Prestataires de services (ménage, plomberie, cours, nounous, coaching)
  • Entreprises locales (restaurants, boutiques)
  • Livreurs internes (employés de LUXESHOP)
  • Admins & équipes internes (support, logistique, finance, marketing)

Cas d'Usage Clés

  • Acheter un produit (neuf, occasion, périssable)
  • Réserver un service (créneaux horaires, prestataire)
  • Trouver/acheter une formation en ligne
  • Commander à manger (resto avec livraison directe par le resto)
  • Proposer un service local (plombier) et être payé via la plateforme

Valeur Ajoutée & Positionnement

Points Forts

  • Tout-en-un : produits + services + prestataires sur la même plateforme
  • Inclusif : particuliers et pros peuvent vendre
  • Livraison interne pour garantir qualité et confiance (sauf restaurants)
  • Support multilingue (français, anglais, langues locales)
  • Outils de visibilité (classement, abonnements premium, sponsorisation)
  • Modèle social : mise en avant des compétences locales

Rôles Utilisateurs & Permissions

Client / Acheteur

  • Inscription obligatoire
  • Profil client, historique commandes, wallet, wishlist, avis

Vendeur (produits)

  • Demande d'inscription + KYC obligatoire (validation manuelle)
  • Mini-boutique: logo, description, politique de livraison, produits
  • Dashboard ventes, stats, gestion stocks, création coupons, promotions

Prestataire de services

  • Profil avec portfolio, tarifs, horaires, géolocalisation, réservation
  • Possibilité d'indiquer disponibilités, options « urgent (24h) »

Livreur (interne)

  • Dashboard courses, historique livraisons, rémunération, notes
  • Attribution des courses via backoffice (ou système d'affectation automatique)

Admin (multi-niveaux)

  • Super-admin : accès complet
  • Admin opérationnel : gestion commandes, litiges
  • Modérateur : validation annonces, modération contenu
  • Comptable / Finance : accès aux transactions et virement
  • Support client : gestion tickets

Restaurant (cas spécial)

  • Boutiques avec option « livraison par le restaurant » (pas besoin de créer compte livreur)

Permissions

  • RBAC (role-based access control) configuré dans Django (groupes + permissions)
  • Logs d'activité pour actions critiques

Architecture Technique & Stack Recommandée

Stack Recommandé

  • Backend : Django (REST API via Django REST Framework pour endpoints)
  • Frontend : Django templates (initial), responsive + option PWA
  • DB : PostgreSQL (production)
  • Cache : Redis (sessions, cache, tâches)
  • Asynchrone / Tasks : Celery + Redis / RabbitMQ
  • Stockage média : S3-compatible (DigitalOcean Spaces / AWS S3) + CDN (Cloudflare)
  • Payments : CinetPay (intégration principale), fallback Stripe/PayPal pour international
  • Authentification : JWT pour API + sessions pour templates. 2FA (optionnel)
  • Logs & Monitoring : Sentry (erreurs), Prometheus/Grafana (metrics), Papertrail/ELK pour logs
  • CI/CD : GitHub Actions → tests → staging → prod
  • Hébergement initial : DigitalOcean droplet (app + DB managed) ou Render/Heroku/Dokku (selon budget)
  • DNS & sécurité : Cloudflare (WAF, CDN, SSL)

Architecture Logique

Client (Web / Mobile) ⇄ Load Balancer/CDN ⇄ Web App (Django) ⇄ Postgres + Redis ⇄ Celery workers

Services externes : CinetPay (paiement), fournisseur de taux de change (ex: exchangeratesapi), service SMS (Infobip/Twilio/Local), Email (SendGrid/Mailgun), Sentry, CDN.

Paiements, Wallet, Escrow & Commissions

Intégration Paiement

  • CinetPay : passerelle principale (Mobile Money + cartes)
  • Stripe / PayPal : option internationale
  • SMS/Email for OTP & confirmation

Wallet Interne

  • Users peuvent déposer de l'argent (via CinetPay) → créditer wallet
  • Paiements par wallet réduisent friction et facilitent remboursements rapides
  • Retraits : KYC requis pour vendeurs

Escrow

  • À la confirmation de paiement : fonds placés en escrow (compte bloqué LUXESHOP)
  • Libération :
    • Automatique : X heures après confirmation de livraison (p. ex. 48h) si pas de dispute
    • Manuelle : si dispute -> admin tranche
    • Exception : pour services, libération après validation du prestataire & client
  • Avantages : réduit chargebacks, protège clients & vendeurs

Commissions & Business Model

  • Inscription des vendeurs : gratuite
  • Publication d'annonce : gratuite (pour volume)
  • Commission sur vente (produits/services) :
    • Vendeur Standard : 5% du montant de la vente
    • Abonné Premium Basic : 3%
    • Abonné Premium Pro : 2%
  • Abonnements Premium (exemples) :
    • Premium Basic : 10 USD / mois → mise en avant + réduction commissions (3%)
    • Premium Pro : 20 USD / mois → forte mise en avant + réduction (2%) + outils marketing
  • Extra : placements sponsorisés & publicités, service fulfillment/stockage, frais de livraison premium

Multi-devises

  • Afficher prix en GNF par défaut
  • Convertir USD/EUR au taux du jour (API de taux) et verrouiller le taux au moment de la commande
